# Supply Chain Shortage Evaluation Framework

## Objective

Develop a reproducible evaluation framework for machine learning models in supply chain shortage prediction.

## Research Questions

- How robust are common ML models under temporal shifts?
- How do models behave under missing data and demand shocks?
- How well do models generalize across categories?

## Planned Baselines

- Logistic Regression
- Random Forest
- XGBoost
- LSTM (optional)

## Evaluation Goals

- Temporal generalization
- Robustness testing
- Cross-category evaluation
- Reproducibility
